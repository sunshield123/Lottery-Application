This project is called Lottery App

The app is written in TypeScript on Angular framework and the backend is written in Node JS on Node.

The Project have following Features

  • When Click Lets Play,Diplays 5 unique Number

  • When Click Draw History,Diplays all previous Play

Backend server is hosted on localhost:3200 and Frontend is running on the localhost:4200
